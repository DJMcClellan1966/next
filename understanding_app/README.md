### **1. Start Backend**

```bash
cd understanding_app

```

Backend runs on http://localhost:8003

### **2. Load Bible Data**

```python
import requests

# Load Bible verses
verses = [
    {
        "reference": "Psalm 23:1",
        "text": "The Lord is my shepherd, I lack nothing.",
        "book": "Psalm",
        "chapter": 23,
        "verse": "1",
        "theme": "trust"
    }
    # ... more verses
]

response = requests.post(
    "http://localhost:8003/api/bible/add-content",
    json=verses
)
```

### **3. Use API**

```python
# Get deep understanding
response = requests.post(
    "http://localhost:8003/api/understanding/generate",
    json={
        "verse_reference": "Psalm 23:1",
        "verse_text": "The Lord is my shepherd, I lack nothing.",
        "depth_level": "deep",
        "scholar_style": "devotional"
    }
)

print(response.json()['understanding'])
```
