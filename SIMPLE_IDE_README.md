### **Basic Usage**

1. **Write Code:**
   - Type Python code in the editor
   - Press F5 to run
   - See output in the output panel

2. **Save/Open Files:**
   - Ctrl+S to save
   - Ctrl+O to open
   - File menu for more options

3. **Check Syntax:**

### **AI Not Working:**
- Check if ML Toolbox is installed
- Check output panel for errors
- Try simpler tasks first

### **Code Not Running:**
- Check syntax first (Check button)
- Look at output panel for errors
- Make sure code is valid Python

### **No Output:**
- Code might be running (check status bar)
- Wait a moment
- Check for errors in output panel
