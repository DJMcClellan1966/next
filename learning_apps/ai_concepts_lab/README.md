# AI Concepts Lab

**Russell & Norvig style: game theory, search & planning, reinforcement learning, probabilistic reasoning.**

Uses **ml_toolbox.ai_concepts** (game_theory, cooperative_games, search_planning, reinforcement_learning, probabilistic_reasoning, clustering).

## Run (from repo root)

```bash
python learning_apps/ai_concepts_lab/app.py
```

Open **http://127.0.0.1:5004**

## Contents

- **Game Theory**: Nash equilibrium, cooperative games
- **Search & Planning**: BFS, DFS, heuristics
- **Reinforcement Learning**: Value functions, Q-learning
- **Probabilistic Reasoning**: Bayesian networks, clustering

Learn by book or level. Demos can be added in demos.py.
