# Python Practice Lab

**Reed & Zelle style: problem decomposition, algorithm patterns, data structures, code organization.**

Uses **reed_zelle_patterns.py** at repo root (ProblemDecomposition, AlgorithmPatterns, DataStructureOptimizer, CodeOrganizer, RecursiveSolutions, IterativeRefinement).

## Run (from repo root)

```bash
python learning_apps/python_practice_lab/app.py
```

Open **http://127.0.0.1:5006**

## Contents

- **Problem Decomposition**: ML pipeline decomposition, algorithm step decomposition
- **Algorithm Patterns**: Divide and conquer, greedy, recursive solutions, iterative refinement
- **Data Structures & Code Org**: Data structure optimizer, code organizer

Learn by book or level. Demos can be added in demos.py.
