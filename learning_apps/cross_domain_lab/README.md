# Cross-Domain Lab

**Unusual cross-domain concepts: quantum mechanics, statistical mechanics, linguistics, precognition, self-organization.**

Uses **ml_toolbox.textbook_concepts** (quantum_mechanics, statistical_mechanics, linguistics, precognition, self_organization).

## Run (from repo root)

```bash
python learning_apps/cross_domain_lab/app.py
```

Open **http://127.0.0.1:5005**

## Contents

- **Quantum Mechanics**: Quantum-inspired ML
- **Statistical Mechanics**: Simulated annealing
- **Linguistics**: Syntactic parsing, grammar-based features
- **Precognition**: Precognitive forecaster, failure modes
- **Self-Organization**: Self-organizing maps, dissipative structures

Learn by book or level. Demos can be added in demos.py.
