## Usage

### 1. AI Code Generation

1. Type your task in the "AI Assistant" input
2. Press Enter or click "Generate"
3. Code is automatically inserted into editor

Example: "Classify data into 3 classes"

### 2. Quick Model Training

1. Select task type (classification/regression/clustering)
2. Click "Train"
3. View results in ML Toolbox panel

### 3. Error Fixing

1. Write code with error
2. Click "AI Fix Error"
3. Error is automatically fixed

### 4. Code Execution

1. Write or generate code
2. Click "Run Code"
3. View results in Results panel
